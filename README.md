# AWS ECS Fargate Docker Application

## Project Overview
Containerized a web application using Docker and deployed it on Amazon ECS using AWS Fargate.

## AWS Services Used
- Amazon ECS
- AWS Fargate
- Amazon ECR
- IAM

## Architecture

Developer
↓
Docker
↓
Amazon ECR
↓
Amazon ECS
↓
AWS Fargate

## Features
- Docker containerization
- Serverless container hosting
- Scalable deployment

## Deployment Steps
1. Build Docker image.
2. Push image to Amazon ECR.
3. Create ECS Cluster.
4. Create Task Definition.
5. Deploy using Fargate.

## Learning Outcomes
- Docker
- ECS
- Amazon ECR
- Fargate
